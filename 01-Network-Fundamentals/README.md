# 01 – Network Fundamentals

High-level reminders for CCNA exam:

## OSI & TCP/IP Models (Lightning Review)

- **OSI Layers (7 → 1)**: Application, Presentation, Session, Transport, Network, Data Link, Physical.
- **TCP/IP Layers (4)**: Application, Transport, Internet, Network Access.
- Remember common mappings (e.g., IP = Layer 3, TCP/UDP = Layer 4, Switch = Layer 2, Router = Layer 3).

## Common Protocols by Layer

- **Layer 2**: Ethernet, 802.1Q VLAN tagging, MAC addresses.
- **Layer 3**: IPv4, IPv6, ICMP, routing protocols.
- **Layer 4**: TCP (connection-oriented), UDP (connectionless).
- **App Examples**: HTTP/HTTPS, DNS, DHCP, SSH, SNMP.

## Key Exam Nuggets

- Collision vs broadcast domains.
- Switch = breaks collision domains, Router = breaks broadcast domains.
- Unicast vs broadcast vs multicast.
